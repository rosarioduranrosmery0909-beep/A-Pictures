# Informe del Proyecto Final

## Nombre del proyecto

A-Pictures

## Descripción

A-Pictures es una aplicación web inspirada en redes sociales como Instagram. Permite a los usuarios registrarse, iniciar sesión y compartir publicaciones con imágenes.

## Funcionalidades

- Registro de usuarios
- Inicio de sesión
- Publicar imágenes
- Dar likes
- Eliminar publicaciones
- Buscar usuarios
- Ver perfiles

## Seguridad implementada

Las rutas del servidor fueron protegidas utilizando Authorization en los headers para evitar acceso no autorizado.

Las imágenes se almacenan en una carpeta separada llamada uploads para proteger los archivos públicos del sistema.

## Base de datos

Se utilizó SQLite para almacenar la información de las publicaciones y usuarios.

## Diseño

La aplicación utiliza una interfaz moderna con colores oscuros y diseño responsive para mejorar la experiencia del usuario.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- Node.js
- Express
- SQLite
